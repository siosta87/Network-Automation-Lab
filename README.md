# Network Automation Security Lab
## Project 1: Basic IP Connectivity Scanner

this tool uses OS library to automate ping requests across a network to verify device availability.

## Project 2: Network Port Security Auditor
A Python based utility that scans specific ports (80,443,22) to audit network security and verify service status.

### Technologies Used
* python 3.x
* Socket & OS Libraries
* Git/GitHub for Version Control
